# System Architecture Wms

- Spring boot 3.0
- Serviçso envolvidos:
  - cadastros
    - armazem
    - loja
    - cliente

    - controle de estoque
    - geração de palete
    - controle de endereços
    - mudança de endereços(pulmão, picking) 
    - doca
